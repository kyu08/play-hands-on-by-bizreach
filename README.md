## @Inject
Google Guice によるDI機能を使用するためのもの。

## ルーティング設定
`conf/routes`の設定に従ってコントローラのメソッドへルーティングされる。

> Point
>
> - マッピング定義で引数の方を省略すると`String`になる。
> - `routes`のコメントに日本語を記述するとコンパイルエラーになることがある。
